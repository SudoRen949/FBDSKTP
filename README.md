# FBDSKTP
A Facebook website embedded in a native window using NeutralinoJS.

# Why NeutralinoJS?
NeutralinoJS is a lightweight web application builder that allows you to create HTML & JS applications and build it to all Operating Systems like Linux, Windows, and MacOS all at once.
NeutralinoJS application consumes less storage compared to Electron based application like Discord or VSCode.
